NOUVELLE PROCEDURE D'AUTHENTIFICATION POUR PARTICIPER AUX EXPERIENCES DU HEC-LABEX!
Pour toute action sur la plateforme ORSEE (p.ex. inscription à une expérience ou modification de vos données sur le profil de participant) vous devez vous authentifier via votre compte SWITCHedu ID.
Comment faire pour vous authentifier avec SWITCH edu-ID ?
  1. Cliquez sur n’importe quel lien d’invitation que vous avez reçu par explab@unil.ch durant ces dernières années;
  2. Vous serez automatiquement dirigé vers la page login/création de compte de SWITCH edu-ID;
    • Créez-vous un compte SWITCHedu ID (si vous n'en avez pas encore);
  3. Connectez-vous à votre compte SWITCHedu-ID;
  4. Si nécessaire, replissez les données manquantes. Veuillez noter que les champs suivants doivent obligatoirement être complétés dans votre compte SWITCH edu-ID : nom, prénom, email, nr. de portable et date de naissance);
  5. Vous serez alors redirigez sur votre profil de participant de la plateforme ORSEE;
  6. Vérifiez et enregistrez vos données sur votre profil de participant. Vous êtes désormais prêt.e à participer aux expériences HEC-LABEX!
